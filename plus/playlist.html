Setup guide

📺 TiviMate (Android / Fire TV)
  1. Download playlist.m3u from the button above.
  2. Open TiviMate → Add playlist → M3U URL.
  3. Enter a name (e.g., "Tree Frog").
  4. For the URL, enter the path to the downloaded file, or upload it to a cloud drive and paste the public link.
📱 IPTV Smarters (iOS / Android)
  1. Download playlist.m3u.
  2. Open IPTV Smarters → Add User → Login with M3U URL.
  3. Provide any username/password (it's ignored) and paste the URL to the playlist.
🖥️ VLC / Kodi / desktop players
  1. Download playlist.m3u.
  2. VLC: Media → Open File…. Kodi: PVR IPTV Simple Client add-on.
